Добрый день!

В данной папке представлено решение на тестовое задание п.1 и п.2.
Решение состоит из
1) папки config, которая содержит файл setting.py с настройками и файл test_config.json с конфигурацией для браузера.
Данный json файл можно дополнять, тем самым изменяя настройки для конкретного браузера.
2) папки test_page, в ней содержатся файлы - описания классов, соответсвующих страницам, которые использовались в тестировании.
Главным классом является BasePage, остальные классы наследуются от него.
3) файл conf_test.py для создания объектов типа webdriver.
4) test1 и test2 - два файла решения по сценарию 1 и сценарию 2.
5) данный текстовый файл readmine.txt

Для решния используются библиотеки: pytest, selenium, bs4, requests, time
